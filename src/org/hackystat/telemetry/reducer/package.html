Implements telemetry reducer extension point. A telemetry reducer is reponsible for processing raw software metrics to generate an instance of TelemetryStreamCollection. The input to a reducer is an array of String objects containing instructions fine tuning reducer behaviors.

This package is part of telemetry infrastructure. It provides a plug-in mechanism whereby new implementation of reducers can be dynamically added to the system. Within the telemetry infrastructure, TelemetryEvaluator uses TelemetryReducerManager as a proxy to resolve reducer call.
