Provides the implemetation of the Telemetry Language reducer construct, which processes sensor data and generates an instance of a TelemetryStreamCollection. The input to a reducer is an array of String objects representing parameters to modify reducer behaviors.

This package is part of telemetry infrastructure. It provides a plug-in mechanism whereby new implementations of reducers can be dynamically added to the system. Within the telemetry infrastructure, TelemetryEvaluator uses TelemetryReducerManager as a proxy to resolve reducer call.
