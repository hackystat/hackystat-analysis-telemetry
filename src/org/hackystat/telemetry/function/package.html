Implements telemetry function extension point. A telemetry function operates on an array of Number, String, and/or TelemetryStreamCollection objects, and returns either a Number or TelemetryStreamCollection object.

This package is part of telemetry infrastructure. An external client normally should not use any classes in this package except supplying implementation of additional funtions. Otherwise, the intended client of this package is TelemetryEvaluator. All function invocation are done through TelemetryFunctionManager proxy.
